# \# AI-Guided Fuzzing Research



This repository contains a research prototype for comparing traditional random fuzzing with AI-guided fuzzing.



\## Research Goal



The goal is to test whether language models can generate smarter and more structured test inputs than random fuzzing.



\## Main Comparison



\- Random fuzzing

\- AI-guided fuzzing



\## Metrics



\- Number of bugs found

\- Time to discover bugs

\- Code coverage



\## Current Scope



The first version will use small Python target programs such as parsers or calculators.
